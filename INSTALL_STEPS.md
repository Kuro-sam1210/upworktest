# Install Component on Your Sandbox

## Direct Links:

**Install Component:**

```
https://mytestsite.discourse.group/admin/customize/themes/components
```

**Or navigate manually:**

1. Admin → Customize → Themes
2. Click **"Components"** tab (at the top)
3. Click **"Install"** button
4. Choose **"From GitHub"**
5. Paste: `https://github.com/AqsaAltaf1/compound-governance-widget`
6. Click **"Install"**

## Then Enable It:

1. Go back to **"Themes"** tab
2. Click on **"Foundation"** (or your active theme)
3. Scroll to **"Included components"**
4. Click **"Select..."** dropdown
5. ✅ Check **"compound-governance-widget"**
6. Click **"Save"**

## Test It:

1. Create a new topic
2. Paste this Tally URL:
   ```
   https://www.tally.xyz/gov/compound/proposal/510?govId=eip155:1:0x309a862bbC1A00e45506cB8A802D1ff10004c8C0
   ```
3. Widget should appear on the right side!
